# Dungeon - Nukkit地下城插件

一个功能丰富的Minecraft Nukkit服务器地下城插件，提供完整的RPG地下城体验。

## 功能特性

### 🏰 地下城系统
- 自动生成的地下城世界
- 智能刷怪机制，根据玩家等级动态调整怪物难度
- 丰富的战利品系统，包含稀有物品掉落

### ⚔️ RPG元素
- 玩家等级系统，支持等级提升和属性成长
- 装备升级系统，可强化武器和防具
- 经验和经济奖励机制

### 🎮 交互功能
- 完整的GUI界面系统
- 自定义命令系统
- 团队合作机制
- 经济系统集成（支持EconomyAPI）

### 🔧 技术特性
- 基于Nukkit插件框架开发
- 支持Maven构建
- 配置文件高度可定制
- 资源包集成

## 安装要求

- Nukkit服务器（API版本：1.0.0）
- Java 17或更高版本
- 依赖插件：
  - EconomyAPI（必需）
  - Tips（可选，用于提示功能）

## 快速开始

### 1. 构建插件
```bash
mvn clean package
```

### 2. 安装插件
1. 将生成的 `Dungeon-1.0-SNAPSHOT.jar` 文件放入服务器的 `plugins` 文件夹
2. 重启服务器
3. 插件会自动生成配置文件

### 3. 配置地下城
编辑 `plugins/Dungeon/config.yml` 文件来自定义：
- 怪物生成设置
- 物品掉落概率
- 经验和经济倍率
- 等级压制参数

## 命令列表

| 命令 | 权限 | 描述 |
|------|------|------|
| `/rpg` | 玩家 | 打开地下城主菜单 |

## 配置文件说明

### 主要配置项
- `command`: 主命令名称（默认："rpg"）
- `地下城世界`: 地下城世界名称（默认："dungeon"）
- `掉落材料`: 怪物掉落的基础材料
- `等级压制倍率`: 高等级玩家对低等级怪物的伤害加成
- `刷怪设置`: 包括刷怪范围、类型、权重等

### 经济系统
- `玩家升级需要经济`: 玩家升级所需金钱
- `装备升级需要经济`: 装备升级所需金钱
- `经济倍率`: 各种经济奖励的倍率

### 经验系统
- `玩家升级需要经验`: 玩家升级所需经验
- `等级击杀经验倍率`: 击杀怪物经验奖励倍率
